
       Change ICQ Details plugin v0.3.3 for Miranda IM
       _______________________________________________


About
_____

This Miranda IM plugin allows you to change your ICQ user details
such as sex, age and location.



Installation
____________

Simply copy "changeinfo.dll" into your Miranda IM plugin directory (usually
"C:\Program Files\Miranda IM\Plugins\") and restart Miranda IM.

You start the import by selecting "Import..." in the main menu.
