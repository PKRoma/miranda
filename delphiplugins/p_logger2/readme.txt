About
-----

Logger2 Plugin 
Version 1.0
by Christian Kästner
for Miranda ICQ 0.1
written with Delphi 5 Pro


Description
-----------

This plugin logs all ICQ activities reported by the ICQLib into 
the miranda.icq.log file. It adds infos like time, duration,
operating system and Miranda version.

Features
--------

+ Logging into a file
+ OS and Miranda version
+ Time and duration of each item in logfile
