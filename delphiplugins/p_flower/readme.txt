About
-----

Flower Plugin
Version 1.0
by Christian Kästner
for Miranda ICQ 0.1
written with Delphi 5 Pro


Description
-----------

Shows a flower on startup and plays a sound as requested by "nobody"
in the forum:

>How about when you start up Miranda ICQ there is a big flower in the 
>middle of the screen, and then a really loud horn noise thats so 
>annoying you just have to change the sound. But make it really hard 
>to change the sound! 
>
>That would be sweet, more people will get miranda if it has this feature. 
>I don't think any other programs have had it. Pure genius

I even extended this and made it impossible to change the sound!


Features
--------

+ Shows the flower
+ Plays a horn sound


Installation
------------

Just copy the dll into Miranda's plugin subdirectory.


Source
------

The full source for this plugin can be found in the Miranda CVS
at Sourceforge.
