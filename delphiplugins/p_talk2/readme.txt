About
-----

Talk2 Plugin 
Version 0.1 Alpha
by Christian Kästner
for Miranda ICQ 0.1
written with Delphi 5 Pro


Description
-----------

Announces online users and reads incoming messages via the Microsoft Text2Speech technology.

Features
--------

+ Announces contact status changes
+ Reads incoming messages
+ Some options

Hints/Comments
--------------

+ This is just an alpha version, still very buggy.

+ A German speech engine can be obtained from logox.de for free (just install the free demo).
